# Flight-Data-Analysis.
Excel Data Analysis project exploring airline flight prices based on airline, stops, duration, and departure time using pivot tables and visualizations.

# ✈️ Airline Flight Price Analysis (Excel Project)

## 📌 Project Overview
This project analyzes airline flight pricing data using Microsoft Excel.  
The goal is to understand how different factors such as airline, number of stops, travel duration, and booking days affect ticket prices.

## 📊 Dataset Information
The dataset contains the following columns:

- Airline
- Flight Number
- Source City
- Destination City
- Departure Time
- Arrival Time
- Stops
- Travel Class
- Duration
- Days Left (before departure)
- Price

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Aggregation

## 📈 Analysis Performed
The following insights were generated:

1. Average price by airline
2. Average price by class
3. Number of flights by airline
4. Average flight duration
5. Impact of number of stops on ticket price

## 📊 Key Insights
- Flights with **more stops tend to have lower prices**.
- **Shorter durations usually cost more** for premium airlines.
- Prices increase as **days left before departure decreases**.
- Some airlines consistently have **higher average ticket prices**.
